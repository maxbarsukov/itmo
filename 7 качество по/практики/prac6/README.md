# Практика 6. Поиск и регистрация дефектов

## Задание

### Задание №1

1. Сделать чек-лист для проверки [формы](https://playground.learnqa.ru/puzzle/triangle);
2. Протестировать форму;
3. Завести дефекты в Google Sheets ([шаблон чек-листа](./Практика%206.%20Шаблон%20чек-листа.docx));
4. Презентовать чек-лист и найденные дефекты.

### Задание №2

1. Изучить [спецификацию](https://qa-ep-bva-practice-assignment.vercel.app/);
2. Протестировать реализации (две реализации в зависимости от группы). Реализация 1 верна, остальные — нет;
3. Завести найденные дефекты в Google Sheets ([шаблон](./Практика%206.%20Шаблон%20отчета%20о%20дефекте.xlsx));
4. Презентовать найденные дефекты.
